# Gesture Prediction Using DQN Based on sEMG Signals

This repository implements a Deep Q-Network (DQN)-based framework for gesture recognition using surface electromyography (sEMG) signals. The project reformulates myoelectric gesture classification as a sequential decision-making problem, where a reinforcement learning agent learns to map raw multi-channel muscle signals to gesture classes.

The goal is to explore whether a DQN agent, supported by a 1D Convolutional Neural Network (1D CNN), can learn movement-intention patterns directly from noisy, high-dimensional sEMG time-series data. This work is motivated by applications in myoelectric control, human-machine interaction, prosthetic control, and rehabilitation-oriented assistive devices.

---

## Project Motivation

Surface electromyography (sEMG) records the electrical activity generated by skeletal muscles. It is widely used in myoelectric interfaces, prosthetic control, rehabilitation devices, and human-machine interaction systems.

However, sEMG signals are difficult to model because they are:

- high-dimensional,
- stochastic,
- non-stationary,
- user-dependent,
- and sensitive to noise, electrode placement, and muscle fatigue.

Traditional gesture-recognition pipelines often rely on handcrafted features such as RMS, MAV, waveform length, or frequency-domain descriptors. In contrast, this project investigates an end-to-end reinforcement learning approach that allows an agent to learn directly from raw sEMG windows.

---

## Key Idea

Instead of treating gesture recognition as a standard supervised classification task, this project frames it as a Markov Decision Process (MDP):

- **State**: a sliding window of recent multi-channel sEMG signals  
- **Action**: predicted gesture class  
- **Reward**: positive reward for correct prediction, negative reward for incorrect prediction  
- **Agent**: a DQN model using a 1D CNN as the Q-value function approximator  

This allows the model to learn gesture-classification policies through trial-and-error interaction with an environment wrapper built from recorded sEMG data.

---

## Method Overview

The full pipeline consists of:

1. **sEMG Data Loading**
   - Uses raw sEMG time-series data.
   - Ground-truth gesture labels are taken from the corresponding stimulus labels.

2. **Sliding Window State Representation**
   - Continuous sEMG signals are segmented into fixed-length temporal windows.
   - Each state contains recent muscle activity across selected channels.

3. **Custom MDP Environment**
   - The static dataset is wrapped as an interactive reinforcement learning environment.
   - The agent receives rewards based on whether its predicted gesture matches the true label.

4. **1D CNN Q-Network**
   - A 1D CNN extracts temporal patterns from multi-channel sEMG windows.
   - Fully connected layers output Q-values for each gesture class.

5. **DQN Training**
   - Uses experience replay to reduce temporal correlation.
   - Uses a fixed target network to stabilize Q-value estimation.
   - Uses epsilon-greedy exploration to balance exploration and exploitation.

6. **Experimental Analysis**
   - Tests the effects of input channel count.
   - Tests the effects of action-space dimensionality.
   - Tests the effects of data volume.
